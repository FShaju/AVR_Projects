# AVR_Projects
Collection of AVR microcontroller projects developed in C using Atmel Studio/Microchip Studio. Includes peripheral interfacing, GPIO, timers, interrupts, UART, SPI, I2C, ADC, PWM, and embedded system applications for learning and development.
## Topics Covered
- GPIO Programming
- Timers and Counters
- External and Pin Change Interrupts
- UART Communication
- SPI Communication
- I2C Communication
- ADC Interfacing
- PWM Generation
- LCD and Sensor Interfacing
## Target Devices
- ATmega328P
## Tools Used
- Microchip Studio
- Ardiuno ISP
